* What is Node JS ?
Node.js is a Javascript runtime (environment) built on Chrome's V8 javascript engine.
Node.js is a Javascript running on the server.
Node.js is a free, open-source, cross-platform JavaScript runtime environment that allows developers to run JavaScript code outside of a web browser.

* History of Nodejs?
In 2009 an idea came in the mind of Ryan Dahl (Google Engineer) that why not run javascript outside the browser, so he took V8 engine and embedded in a C++ program and called it Node.exe later on became Nodejs.
